# 脱口秀评分规则（高山交通电台版）

> 马驰 · Don't Die 长寿从运动系统健康开始 · 评委评分标准

---

## 评分维度（5项 × 20分 = 100分）

| # | 维度 | 权重 | 核心考察点 |
|---|------|------|------------|
| 1 | **笑点设计** | 20分 | 段子是否好笑、节奏是否到位、预期违背是否有效 |
| 2 | **结构与节奏** | 20分 | 开场钩子、Part过渡、情绪曲线、时间控制 |
| 3 | **内容价值** | 20分 | 干货密度、数据可信度、观点独特性 |
| 4 | **表达呈现** | 20分 | 语言节奏、肢体语言、临场反应、停顿设计 |
| 5 | **整体印象** | 20分 | 记忆点、金句、情感共鸣、收尾力度 |

### 分档标准

| 等级 | 分数段 | 描述 |
|------|--------|------|
| S | 95-100 | 教科书级别，可作为培训教材 |
| A | 85-94 | 优秀，有1-2处小瑕疵 |
| B | 75-84 | 良好，有明显提升空间 |
| C | 60-74 | 合格，需要较大修改 |
| D | <60 | 不合格，需重写 |

### 各维度评分细则

#### 1️⃣ 笑点设计（20分）

| 分档 | 描述 |
|------|------|
| 18-20 | 笑点密集且自然，预期违背精准，callback呼应完美，每3-4分钟有笑声高峰 |
| 14-17 | 有明显笑点，但密度或节奏略有不足 |
| 10-13 | 笑点生硬或过于刻意，铺垫过长 |
| <10 | 几乎没有有效笑点，或笑点冒犯感过强 |

#### 2️⃣ 结构与节奏（20分）

| 分档 | 描述 |
|------|------|
| 18-20 | 开场钩子有力，Part之间转场自然，情绪曲线清晰，15分钟刚好饱满 |
| 14-17 | 结构完整但转场略生硬，或某Part略显冗长/仓促 |
| 10-13 | 结构扁平，缺乏起伏 |
| <10 | 逻辑混乱，高潮和收尾乏力 |

#### 3️⃣ 内容价值（20分）

| 分档 | 描述 |
|------|------|
| 18-20 | 干货充足、数据可信、观点独特、让人"原来如此" |
| 14-17 | 有干货但密度不够，或深度略浅 |
| 10-13 | 以感慨为主，缺乏具体知识 |
| <10 | 内容空洞或错误频出 |

#### 4️⃣ 表达呈现（20分）

| 分档 | 描述 |
|------|------|
| 18-20 | 语气自然、重音精准、停顿到位、肢体自信、临场应变能力强 |
| 14-17 | 表达流畅但略显紧张，或重音/停顿不够精准 |
| 10-13 | 背稿感明显，节奏单一 |
| <10 | 语速过快或过慢，完全没有节奏设计 |

#### 5️⃣ 整体印象（20分）

| 分档 | 描述 |
|------|------|
| 18-20 | 有让人记住的金句，情感共鸣强，结束后有"想转发"的冲动 |
| 14-17 | 有记忆点但不够深刻 |
| 10-13 | 听完即忘，无共情 |
| <10 | 内容或表达让人不适 |
